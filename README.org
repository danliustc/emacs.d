#+TITLE: Emacs 配置说明（使用手册）

这是让 Codex 配置的轻量、稳定、Vim-like 个人 Emacs 配置。

适用场景：
- 日常写作：Org/Markdown
- 轻量编程阅读：Python/C++
- 平台：macOS 优先

核心原则：
- 使用 =evil= + Space 风格按键体系（全局 =SPC=，局部 =,=）
- 配置保持精简，编辑器不是 IDE
- 以 literate config（=config.org=）作为唯一主配置入口

* 目录结构

- [[./init.el][init.el]]：入口，负责加载 =config.org=
- [[./config.org][config.org]]：主配置（会 tangle 成 =config.el=）
- [[./tests/config-tests.el][tests/config-tests.el]]：ERT 回归测试
- [[./doc/PROJECT_REQUIREMENTS.md][doc/PROJECT_REQUIREMENTS.md]]：项目协作要求与设计规范（合并）

* 快速开始

** 依赖

- Emacs（建议 29+）
- 可访问 GNU ELPA / MELPA（用于自动安装包）

** 首次启动

启动 Emacs 后会自动执行：
1. 安装缺失包（由 =my/auto-install-packages= 控制）
2. 将 [[./config.org][config.org]] tangle 成 =config.el=
3. 加载 =config.el=

主要包：
- evil
- key-chord
- general
- which-key
- projectile
- markdown-mode
- ayu-theme
- vertico
- orderless
- consult
- marginalia
- embark
- embark-consult

* 功能总览

** 编辑体验

- Vim 模式：启用 =evil-mode=
- 插入态快速退出：输入 =jk=
- 行号默认开启
- 关闭 menu/tool/scroll bar
- 简洁 mode-line（不依赖 doom-modeline）
- 现代补全栈：Vertico + Orderless + Consult + Marginalia + Embark

** 主题与字体

- 主题：=ayu-light=
- 字号：默认高度 =140=（比默认大一档）

** 项目与文件

- recentf 已启用（最近文件）
- projectile 已启用（项目管理）
- =SPC p p= 项目切换（Projectile）

** Org/Markdown

- Org：缩进、强调符隐藏、自定义折叠符号
- Org 折叠：=TAB -> org-cycle=，=<backtab> -> org-shifttab=
- Markdown：自动识别 =.md= / =.markdown=
- Idea 捕获：=SPC a o c i= 统一写入 =~/code/orgfiles/ideas.org=
- 日记捕获：=SPC a o c j= 统一写入 =~/code/orgfiles/journal/=
- 会议笔记捕获：=SPC a o c m= 选择类型并实例化模板

* 快捷键参考

** 全局 Leader（=SPC=）

- =SPC f f= 打开或新建文件
- =SPC SPC= 命令面板（M-x）
- =SPC f r= 最近文件（Consult）
- =SPC f g= 项目内全文检索（ripgrep）
- =SPC f l= 当前 buffer 行内搜索
- =SPC f s= 保存
- =SPC a o a= Org Agenda
- =SPC a o c= Org Capture
- =SPC a o c i= 快速捕获 idea（单文件）
- =SPC a o c j= 快速捕获日记（按天文件）
- =SPC a o c m= 新建会议笔记（目前支持 meeting）
- =SPC a o l= Org Store Link
- =SPC a o t= Org Todo List
- =SPC b b= Consult buffer 列表
- =SPC b s= 原生 buffer 切换
- =SPC b d= 关闭当前 buffer
- =SPC p p= 项目切换
- =SPC p f= 项目内找文件
- =SPC p r= 项目最近文件
- =SPC p d= 打开项目 Dired 列表
- =SPC w /= 垂直分屏
- =SPC w -= 水平分屏
- =SPC w d= 关闭窗口
- =SPC q q= 退出 Emacs
- =SPC h f= 查看函数帮助
- =SPC h v= 查看变量帮助
- =SPC h k= 查看按键帮助
- =C-s= Consult 行搜索
- =C-x b= Consult buffer 切换
- =C-.= Embark 动作菜单
- =C-;= Embark DWIM 动作
- =C-h B= Embark 按键绑定帮助

** 局部 Leader（=,=）

- Org
  - =, t= TODO 切换
  - =, a= Agenda
  - =, s= Schedule
- Markdown
  - =, p= 预览
  - =, h= 插入标题
- Prog（Python/C++等）
  - =, g= 跳转定义
  - =, r= 查找引用

* macOS 约定

仅在 macOS 生效：
- =mac-command-modifier -> super=
- =mac-option-modifier -> meta=

目的：
- 保持 Emacs/Vim 风格操作一致性
- 同时兼容 Mac 键盘习惯

* 开发与测试

运行回归测试：

#+begin_src sh
emacs --batch -l tests/config-tests.el -f ert-run-tests-batch-and-exit
#+end_src

* 会议笔记工作流

- 入口：=SPC a o c m=
- 类型：先选择 =meeting=
- 名称：输入会议名（自动补 =.org= 后缀）
- 目录：=~/code/orgfiles/meetings/=
- 行为：文件不存在时自动写入模板；存在时直接打开

* Idea / 日记工作流

- Idea 入口：=SPC a o c i=
- Idea 文件：=~/code/orgfiles/ideas.org=（统一写入 =Inbox=）
- 日记入口：=SPC a o c j=
- 日记目录：=~/code/orgfiles/journal/=
- 日记文件：按日期自动创建，如 =2026-02-27.org=

如果只验证配置可加载：

#+begin_src sh
emacs --batch -l init.el
#+end_src
