#+TITLE: Emacs 配置说明（使用手册）

这是一个轻量、稳定、Vim 优先的个人 Emacs 配置。

适用场景：
- 日常写作：Org/Markdown
- 轻量编程阅读：Python/C++
- 平台：macOS 优先

核心原则：
- 使用 =evil= + Space 风格按键体系（全局 =SPC=，局部 =,=）
- 配置保持精简，不走重型 IDE 路线
- 以 literate config（=config.org=）作为唯一主配置入口

* 目录结构

- [[./init.el][init.el]]：入口，负责加载 =config.org=
- [[./config.org][config.org]]：主配置（会 tangle 成 =config.el=）
- [[./tests/config-tests.el][tests/config-tests.el]]：ERT 回归测试
- [[./doc/PROJECT_REQUIREMENTS.md][doc/PROJECT_REQUIREMENTS.md]]：项目协作要求与设计规范（合并）

* 快速开始

** 依赖

- Emacs（建议 29+）
- 可访问 GNU ELPA / MELPA（用于自动安装包）

** 首次启动

启动 Emacs 后会自动执行：
1. 安装缺失包（由 =my/auto-install-packages= 控制）
2. 将 [[./config.org][config.org]] tangle 成 =config.el=
3. 加载 =config.el=

主要包：
- evil
- key-chord
- general
- which-key
- projectile
- markdown-mode
- ayu-theme

* 功能总览

** 编辑体验

- Vim 模式：启用 =evil-mode=
- 插入态快速退出：输入 =jk=
- 行号默认开启
- 关闭 menu/tool/scroll bar
- 简洁 mode-line（不依赖 doom-modeline）

** 主题与字体

- 主题：=ayu-light=
- 字号：默认高度 =130=（比默认大一档）

** 项目与文件

- recentf 已启用（最近文件）
- projectile 已启用（项目管理）
- =SPC p p= 显示“最近 5 个项目”候选

** Org/Markdown

- Org：缩进、强调符隐藏、自定义折叠符号
- Org 折叠：=TAB -> org-cycle=，=<backtab> -> org-shifttab=
- Markdown：自动识别 =.md= / =.markdown=

* 快捷键参考

** 全局 Leader（=SPC=）

- =SPC f f= 打开文件
- =SPC f r= 最近文件列表
- =SPC f s= 保存
- =SPC b b= 切换 buffer
- =SPC b d= 关闭当前 buffer
- =SPC p p= 最近 5 个项目切换
- =SPC p f= 项目内找文件
- =SPC p d= 打开项目 Dired 列表
- =SPC w /= 垂直分屏
- =SPC w -= 水平分屏
- =SPC w d= 关闭窗口
- =SPC q q= 退出 Emacs
- =SPC h f= 查看函数帮助
- =SPC h v= 查看变量帮助
- =SPC h k= 查看按键帮助

** 局部 Leader（=,=）

- Org
  - =, t= TODO 切换
  - =, a= Agenda
  - =, s= Schedule
- Markdown
  - =, p= 预览
  - =, h= 插入标题
- Prog（Python/C++等）
  - =, g= 跳转定义
  - =, r= 查找引用

* macOS 约定

仅在 macOS 生效：
- =mac-command-modifier -> super=
- =mac-option-modifier -> meta=

目的：
- 保持 Emacs/Vim 风格操作一致性
- 同时兼容 mac 键盘习惯

* 开发与测试

运行回归测试：

#+begin_src sh
emacs --batch -l tests/config-tests.el -f ert-run-tests-batch-and-exit
#+end_src

如果只验证配置可加载：

#+begin_src sh
emacs --batch -l init.el
#+end_src
