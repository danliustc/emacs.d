#+TITLE: Keymaps Reference

完整快捷键参考。README 只保留了高频入口，这里放全量映射。

* Global Leader (=SPC=)

- =SPC SPC= -> =execute-extended-command=
- =SPC a o a= -> =org-agenda=
- =SPC a o c= -> capture prefix (sub-keys: =t= todo / =b= brainstorm / =i= idea / =j= journal / =m= meeting / =c= dispatch)
- =SPC a o c b= -> =my/orgfiles-capture-brainstorm=
- =SPC a o c c= -> =my/orgfiles-capture-dispatch=
- =SPC a o c i= -> =my/orgfiles-capture-idea=
- =SPC a o c j= -> =my/orgfiles-capture-journal=
- =SPC a o c m= -> =my/orgfiles-capture-meeting=
- =SPC a o c t= -> =my/orgfiles-capture-todo=
- =SPC a o l= -> =org-store-link=
- =SPC a o t= -> =org-todo-list=
- =SPC a d= -> =my/orgfiles-capture-dispatch= (shortcut for =SPC a o c c=)
- =SPC f f= -> =find-file=
- =SPC f g= -> =my/consult-ripgrep= (=consult-ripgrep=, fallback to =rgrep=)
- =SPC f l= -> =my/consult-line= (=consult-line=, fallback to =isearch-forward=)
- =SPC f r= -> =my/consult-recent-file= (=consult-recent-file=, fallback to =recentf-open-files=)
- =SPC f s= -> =save-buffer=
- =SPC b b= -> =my/consult-buffer= (=consult-buffer=, fallback to =switch-to-buffer=)
- =SPC b d= -> =kill-current-buffer=
- =SPC p p= -> =projectile-switch-project=
- =SPC p f= -> =projectile-find-file=
- =SPC p r= -> =projectile-recentf=
- =SPC p d= -> =projectile-dired=
- =SPC w h= -> =windmove-left=
- =SPC w j= -> =windmove-down=
- =SPC w k= -> =windmove-up=
- =SPC w l= -> =windmove-right=
- =SPC w /= -> =split-window-right=
- =SPC w -= -> =split-window-below=
- =SPC w d= -> =delete-window=
- =SPC q q= -> =save-buffers-kill-terminal=
- =SPC z -= -> =my/font-size-decrease=
- =SPC z == -> =my/font-size-increase=
- =SPC z 0= -> =my/font-size-reset=
- =SPC h f= -> =describe-function=
- =SPC h v= -> =describe-variable=
- =SPC h k= -> =describe-key=

* Global Non-Leader Keys

- =C-s= -> =consult-line=
- =C-x b= -> =consult-buffer=
- =C-.= -> =embark-act=
- =C-,= -> =embark-dwim=
- =C-h B= -> =embark-bindings=

* Minibuffer Actions (Consult/Embark Flow)

- =C-c C-o= -> =embark-export= (exports minibuffer results to a buffer)

* Ergonomics Notes

- Line numbers are enabled in =prog-mode= only.
- Writing modes keep visual wrapping enabled and avoid global line-number noise.
- Insert-state escape chord is =fd= with =0.2s= timeout.

* Local Leader (=,=)

** Org Mode
- =, t= -> =org-todo=
- =, a= -> =org-archive-subtree=
- =, s= -> =org-schedule=
- =, r= -> =org-refile=

** Markdown Mode
- =, p= -> =markdown-preview=
- =, h= -> =markdown-insert-header-dwim=

** Prog Mode
- =, g= -> =xref-find-definitions=
- =, r= -> =xref-find-references=
