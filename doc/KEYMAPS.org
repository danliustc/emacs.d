#+TITLE: Keymaps Reference

完整快捷键参考。README 只保留高频入口，这里放全量映射。

* Global Keys

- =M-x= -> =execute-extended-command=
- =C-x C-f= -> =find-file=
- =C-x C-s= -> =save-buffer=
- =C-x b= -> =consult-buffer=
- =C-s= -> =consult-line=
- =C-.= -> =embark-act=
- =C-;= -> =embark-dwim=
- =C-h B= -> =embark-bindings=
- =M-y= -> =consult-yank-pop=
- =M-w= -> =kill-ring-save= (copy active region)
- =C-y= -> =yank= (paste)

* Clipboard

- GUI Emacs uses the system clipboard through =select-enable-clipboard=.
- Terminal Emacs remaps:
  - =M-w= -> =kill-ring-save=, then =pbcopy=
  - =C-w= -> =kill-region=, then =pbcopy=
  - =C-y= -> =my/pbpaste=
- =M-y= -> =consult-yank-pop=

* Personal =C-c= Keys

- =C-c a= -> =org-agenda=
- =C-c c= -> =org-capture=
- =C-c l= -> =org-store-link=
- =C-c s= -> =my/search-ripgrep= (=consult-ripgrep=, fallback to =rgrep=)
- =C-c f= -> =my/find-file-by-name= (=consult-find=, fallback to =find-file=)
- =C-c o= -> =my/jump-outline= globally (=consult-outline=, fallback to =imenu=); =consult-org-heading= in =org-mode= when =consult= is installed
- =C-c m= -> =consult-mark=
- =C-c e= -> =consult-flymake=
- =C-c t= -> =my/toggle-file-sidebar= (=dired-sidebar-toggle-sidebar=, fallback to =dired=)
- =C-c b= -> =my/switch-buffer= (=consult-buffer=, fallback to =switch-to-buffer=)

* GTD Navigation Keys

- =C-c g i= -> =my/open-inbox=
- =C-c g p= -> =my/open-personal=
- =C-c g w= -> =my/open-work=
- =C-c g c= -> =my/open-config=
- =C-c g d= -> =my/gtd-daily-review=
- =C-c g n= -> =my/gtd-process-inbox=

* Minibuffer Actions (Consult/Embark Flow)

- =C-.= -> =embark-act=
- =C-;= -> =embark-dwim=

* Org Mode

- =TAB= -> =org-cycle=
- =<backtab>= -> =org-shifttab=
- =C-c C-w= -> =org-refile= (Org default)
- =C-c C-t= -> =org-todo= (Org default)
- =C-c C-s= -> =org-schedule= (Org default)
- =C-c C-x C-a= -> =org-archive-subtree= (Org default)

* Org Agenda

- =q= -> =org-agenda-quit=
- =C-c a d= -> daily dashboard custom command
- =C-c a W= -> weekly overview custom command
- =C-c a p= -> personal tasks custom command
- =C-c a k= -> work tasks custom command

* Markdown Mode

- =C-c C-p= -> =markdown-preview=
- =C-c C-h= -> =markdown-insert-header-dwim=

* Prog Mode

- Line numbers are enabled in =prog-mode= via =display-line-numbers-mode=.

* Ergonomics Notes

- Evil/Vim modal editing is disabled.
- There is no =fd= escape chord.
- Line numbers are enabled in =prog-mode= only.
- Writing modes keep visual wrapping enabled and avoid global line-number noise.
