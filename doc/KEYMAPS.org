* Global Leader (=SPC=)

- =SPC SPC= -> =execute-extended-command=
- =SPC a o a= -> =org-agenda=
- =SPC a o b= -> =my/orgfiles-capture-brainstorm=
- =SPC a o c= -> =my/orgfiles-capture-dispatch=
- =SPC a o l= -> =org-store-link=
- =SPC a o t= -> =org-todo-list=
- =SPC a .= -> =embark-act=
- =SPC a ,= -> =embark-dwim=
- =SPC f f= -> =find-file=
- =SPC f g= -> =consult-ripgrep=
- =SPC f l= -> =consult-line=
- =SPC f r= -> =consult-recent-file=
- =SPC f s= -> =save-buffer=
- =SPC b b= -> =consult-buffer=
- =SPC b d= -> =kill-current-buffer=
- =SPC p p= -> =projectile-switch-project=
- =SPC p f= -> =projectile-find-file=
- =SPC p r= -> =projectile-recentf=
- =SPC p d= -> =projectile-dired=
- =SPC w h= -> =windmove-left=
- =SPC w j= -> =windmove-down=
- =SPC w k= -> =windmove-up=
- =SPC w l= -> =windmove-right=

- =SPC w -= -> =split-window-below=
- =SPC w d= -> =delete-window=
- =SPC q q= -> =save-buffers-kill-terminal=
- =SPC z -= -> =my/font-size-decrease=
- =SPC z == -> =my/font-size-increase=
- =SPC z 0= -> =my/font-size-reset=
- =SPC h f= -> =describe-function=
- =SPC h v= -> =describe-variable=

* Global Non-Leader Keys

- =C-s= -> =consult-line=
- =C-x b= -> =consult-buffer=
- =C-.= -> =embark-act=
- =C-,= -> =embark-dwim=
- =C-h B= -> =embark-bindings=
- =C-c p p= -> =projectile-switch-project= (fallback: =project-switch-project=)
- =C-c p f= -> =projectile-find-file= (fallback: =project-find-file=)

* Minibuffer Actions (Consult/Embark Flow)

- =C-.= -> =embark-act=
- =C-,= -> =embark-dwim=
- =C-c C-o= -> =embark-export=

* Ergonomics Notes

- Line numbers are enabled in =prog-mode= only.
- Writing modes keep visual wrapping enabled and avoid global line-number noise.
- Insert-state escape chord is =fd= with =0.2s= timeout.

* Local Leader (=,=)
